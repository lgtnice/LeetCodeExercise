

# 链表

|            | 时间复杂度 | 空间复杂度 |
| ---------- | ---------- | ---------- |
| 寻址、修改 | O(n)       | O(1)       |
| 插入       | O(1)       | O(1)       |
| 删除       | O(1)       | O(1)       |
|            |            |            |

根据链表和数组的时间复杂度。数组适合数据固定、频繁查询的场景；链表适合数据不固定，频繁增删的场景。

## 虚拟头结点

使用虚拟头结点可以以一种更简洁的方法删除单链表的头部结点。
